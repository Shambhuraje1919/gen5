# Contributing to GEN5

Thank you for your interest in contributing! We welcome contributions in the form of bug fixes, new features, tests, and documentation improvements.

## How to Contribute

1. **Fork the repository** and clone it locally:

   ```bash
   git clone https://github.com/<your-username>/gen5.git
   cd gen5
   ```
2. **Create a branch** for your changes:

   ```bash
   git checkout -b feature/my-feature
   ```
3. **Install dependencies**:

   ```bash
   pip install -r gen5/requirements.txt
   pip install -r dev-requirements.txt
   ```
4. **Run tests** to make sure everything works:

   ```bash
   pytest
   ```
5. **Make your changes**, following code style and documentation conventions.
6. **Commit your changes** with a clear message:

   ```bash
   git commit -m "Add feature XYZ"
   ```
7. **Push your branch** to your fork and open a Pull Request against `master`.

## Code of Conduct

Please follow respectful and constructive behavior in all communications.

## What We Accept

* Bug fixes
* New features
* Test improvements
* Documentation updates

## Notes

* Ensure your code passes all tests locally.
* Add tests for new functionality wherever applicable.
* Keep pull requests focused and concise.
